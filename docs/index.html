NeuralFlow is a local-first desktop app for orchestrating AI agents visually. Drag nodes, connect them, pick any model, and run — no Docker, no cloud account, no Python degree required.

Get Started View on GitHub
NeuralFlow — research-assistant.nf

Everything you need to build AI workflows

From simple chains to complex multi-agent hierarchies — all visual, all local.

🎨

Visual Canvas

Infinite drag-and-drop canvas powered by React Flow. 10 node types, typed edges, auto-layout, undo/redo, mini-map, and a full properties panel for inline editing.

🤖

Any Model, Any Provider

OpenAI, Anthropic, DeepSeek, Groq, Mistral, Gemini, AWS Bedrock, Ollama (local), LM Studio, and any OpenAI-compatible API — all via LiteLLM.

🐛

Step-through Debug Replay

Load any past run, inspect every LLM call and tool result, and re-run from any step. Full run history with status, duration, and cost per node.

🧠

Memory & RAG

Memory nodes with a full RAG pipeline: document chunking, embedding, and semantic search backed by sqlite-vec. No external vector database needed.

💰

Cost Dashboard

Per-node, per-agent, per-run cost breakdown from LiteLLM usage data. SVG charts by day, workflow, and model. CSV export and configurable cost alerts.

🔌

Plugin System

Install custom node types and tools via Python packages with neuralflow_plugins entry points. Hot-reload plugins without restarting the app.

Up and running in 3 commands

Prerequisites: Rust, Node.js 20+, pnpm, Python 3.11+, uv.

# 1. Clone
git clone https://github.com/ssnelavala-masstcs/neuralflow
cd neuralflow

# 2. Install all dependencies
./scripts/setup-dev.sh

# 3. Launch (opens the desktop app automatically)
./scripts/dev.sh
