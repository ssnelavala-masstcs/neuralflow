Core Capabilities

// Everything you need to build, run, and monitor AI agent workflows

🧠

Multi-Agent Orchestration

Drag Agent, Router, and Human nodes onto a canvas. Connect them with directed edges. Run sequentially, hierarchically, or as a state machine.

🔌

Any Model, Any Provider

OpenAI, Anthropic, Groq, Mistral, Gemini, Ollama, AWS Bedrock — swap models per-node via LiteLLM. No vendor lock-in.

💾

Local-First & Private

All data stays on your machine. SQLite workspace storage, OS keychain for API keys. No cloud dependency.

🔍

Built-In RAG & Memory

Ingest documents, embed with sqlite-vec, and search semantically — no external vector database required.

🛠️

MCP Tool Integration

Connect to any Model Context Protocol server. Give agents filesystem, web, and custom tool access.

📊

Cost Tracking

Every inference call is logged with token counts and USD cost. Set alerts and export analytics.

Quick Start

// Get running in under 2 minutes

# 1. Clone
git clone https://github.com/ssnelavala-masstcs/neuralflow.git
cd neuralflow

# 2. Install sidecar + frontend deps
pip install uv
cd packages/sidecar && uv sync && cd ../..
npm install -g pnpm
cd apps/desktop && pnpm install && cd ../..

# 3. Run
./scripts/dev.sh
